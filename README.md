# Super Mario JavaScript Game

## Overview
A classic 2D Mario-style platformer game developed using HTML, CSS, and JavaScript. This project demonstrates core game development concepts such as character movement, collision detection, animations, and game physics.

## Features
- Mario-style gameplay
- Character movement and jumping
- Collision detection
- Responsive game controls
- 2D platform mechanics

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Canvas API

## Skills Demonstrated
- Frontend Development
- Game Logic Implementation
- JavaScript Programming
- Event Handling
- Animation & Rendering

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/manswisahare/mario-game.git
```

2. Open `index.html` in your browser.
